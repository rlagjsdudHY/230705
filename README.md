# 230705
- Spring Boot 팀프로젝트 (팀원 4명)
- Spring Boot로 구현한 컴퓨터 판매 게시판 입니다.

- sts4 버전 2.7.4
- Java 11
- MySQL DB
- MyBatis
- MVC패턴
- 
- 회원관리 - 회원가입,회원탈퇴,회원정보수정,로그인,로그아웃 구현
- 게시판   - 글쓰기,수정,삭제,파일첨부,댓글,비밀글 기능,조회수,페이징 구현
- 어드민   - 어드민로그인,어드민페이지,공지사항,QNA게시판 조회,회원관리기능
